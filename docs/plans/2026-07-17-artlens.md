# ArtLens — Fine-Art Reverse-Image Search Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (inline) — frontend task may be delegated to a subagent. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Deployed reverse-image search over ~8k CC0 Art Institute of Chicago artworks: static React frontend + FastAPI/OpenCLIP backend on Railway.

**Architecture:** Offline `build_index.py` fetches AIC public-domain works, downloads IIIF images (cached on disk, resumable), embeds with OpenCLIP ViT-L-14/laion2b_s32b_b82k, writes `embeddings.npy` (N×768 f32, L2-normed) + `metadata.jsonl` committed to the repo. FastAPI loads model + matrix at startup; `/search` embeds the upload and does exact cosine (matmul) — no ANN below ~100k vectors. Frontend is Vite/React/TS/Tailwind + shadcn-style components served by Caddy on a second Railway service.

**Tech Stack:** Python 3.12, open_clip_torch (CPU torch), FastAPI, uvicorn, numpy, Pillow; React+Vite+TS+Tailwind, react-dropzone; Docker; Railway (2 services); GitHub.

## Global Constraints

- Model locked: OpenCLIP `ViT-L-14`, pretrained `laion2b_s32b_b82k`. Downgrade path: `ViT-B-32` ONLY if Railway size/RAM/cold-start is painful — then rebuild index (image cache makes re-embed cheap). Never mix index/query models.
- All embeddings L2-normalized; similarity = inner product.
- Exact brute-force search; leave comment noting ANN threshold ~100k.
- AIC API: no key; `is_public_domain=true`; skip null `image_id`; plain `/artworks` listing (search endpoint caps at page*limit≤10000); IIIF `{iiif}/{image_id}/full/843,/0/default.jpg` (index) and `200,` (thumbs); descriptive User-Agent; ≤8 concurrent; backoff on 429/5xx.
- Backend: read `$PORT`; CORS from `FRONTEND_ORIGIN` env (not `*`); upload cap 10 MB; `/healthz` → 200.
- Frontend: API base from `VITE_API_URL`; no hardcoded URLs; drag-drop + click, query preview, skeleton loading, results grid w/ score badge + artic.edu link, empty/error states, mobile-friendly.
- Repo: monorepo `/backend`, `/frontend`; artifacts committed (git LFS only if >100 MB — 8k×768×4B ≈ 24 MB, so plain git).
